# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating, you agree to uphold a welcoming, inclusive, and respectful environment for everyone.

## Reporting

If you experience or witness unacceptable behavior, please report it by emailing the project maintainer. All reports will be handled with discretion.

## Enforcement

Project maintainers are responsible for clarifying standards and taking appropriate corrective action in response to unacceptable behavior.

For the full text, see https://www.contributor-covenant.org/version/2/1/code_of_conduct/
